# QuizLounge (Backend)

Das ist ein Semesterprojekt, das im Rahmen des Moduls "Enterprise-Anwendungen" entstanden ist.

Es handelt sich um eine Webanwendung, mit der man Quizze erstellen, bearbeiten, lösen und mit Freunden teilen kann. So wird die Prüfungsvorbereitung nicht so langweilig, wie mit Zusammenfassungen. :)

## Eingebaute Features

-  Fragen mit variablen Antwortmöglichkeiten
-  Fragen mit variablen richtigen Antwortmöglichkeiten
-  Fragen mit Eingabefeld
-  Kategorisieren
-  Statistiken
-  Evaluation der Fragen mit Eingabefeldern mittels OpenAI
-  Freundschaften
